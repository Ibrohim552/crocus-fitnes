# Fitness Club API

## Описание

Это API для управления системой фитнес-клуба. Он предоставляет функционал для управления членами клуба, тренерами, сессиями и бронированиями.

## Технологии

- .NET 8
- C#
- AutoMapper
- Entity Framework (если используется)
- ASP.NET Core

Эндпоинты
Члены
GET /api/members - Получить список всех членов.
GET /api/members/{id} - Получить информацию о члене по ID.
POST /api/members - Добавить нового члена.
PUT /api/members/{id} - Обновить информацию о члене.
DELETE /api/members/{id} - Удалить члена.
Тренеры
GET /api/trainers - Получить список всех тренеров.
GET /api/trainers/{id} - Получить информацию о тренере по ID.
POST /api/trainers - Добавить нового тренера.
PUT /api/trainers/{id} - Обновить информацию о тренере.
DELETE /api/trainers/{id} - Удалить тренера.
Сессии
GET /api/sessions - Получить список всех сессий.
GET /api/sessions/{id} - Получить информацию о сессии по ID.
POST /api/sessions - Добавить новую сессию.
PUT /api/sessions/{id} - Обновить информацию о сессии.
DELETE /api/sessions/{id} - Удалить сессию.
Бронирования
GET /api/bookings - Получить список всех бронирований.
GET /api/bookings/{id} - Получить информацию о бронировании по ID.
POST /api/bookings - Создать новое бронирование.
PUT /api/bookings/{id} - Обновить информацию о бронировании.
DELETE /api/bookings/{id} - Удалить бронирование.
